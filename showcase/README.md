# AI Power Pack v2.5 - Showcase Report
## 框架升级验证：API-First + GUI + 可打包性

本目录包含了根据最新框架规则（v2.5）重新设计的 5 个实战案例。每个案例均具备：
1. **API核心**：所有功能均可通过外部代码控制。
2. **编程SDK**：内置 Python SDK 供自动化调用。
3. **图形界面 (GUI)**：提供 PySide6 交互界面。
4. **自测逻辑**：包含全流程自测脚本。
5. **符合规范**：单文件严格控制在 500 行以内。
